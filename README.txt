In a few simple steps you can start accepting credit card payments with Stripe Checkout on your WordPress site.

== Description ==

WPFormify plugin offers several powerful features to fit your needs. If you’re looking for Stripe Payment Gateway for your WordPress site, get WP Formify and start accepting online credit card payments. It generates dynamic buttons using shortcodes that enable payment form on your WordPress site using Stripe payment gateway. WP Formify works out perfect for accepting simple and donation payment. Fulfill your requirement of one time payments to keep your business going with ease.

== PlUGIN FEATURES ==

* Collect donation and **simple payment** from single button.
* User can enter custom amount to make payment for donation.
* Sandbox support is available for developer testing.
* Notification email will send to admin on successful transaction.

== How To Use ==

* Main admin menu item: Stripe account Settings.
* Second sub menu item is button settings page.
* When payment success email will send to notification email.
* When using for Donation check on user set a Donation amount otherwise leave unchecked to collect fixed amount

== PREMIUM FEATURES ==

* Easy to setup.
* Multiple payment options (Simple, Donation).
* Authentication functionality to connect Stripe.
* Shortcode Supported.
* Global settings option for Stripe.
* Allow fixed or a custom amount.

* Separate notification email for Admin and User.

== Installation ==

1. Upload the `WP Formify` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create a Stripe account. If you don't have an account, go to https://dashboard.stripe.com/register to create one. Register your application with Stripe.
